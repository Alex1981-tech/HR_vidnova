# UI reference structure

Дата: 2026-06-24  
Источник: скриншоты в `docs/фото/`

## Цель

Интерфейс HR Vidnova должен структурно напоминать текущий рабочий HR-портал
Vidnova/PeopleForce: постоянная навигационная оболочка, плотные рабочие экраны,
табличные списки, календарные сетки, hover/detail карточки и быстрые действия в
верхней панели.

Это не маркетинговый dashboard. Первый экран после входа остается рабочей
домашней страницей с задачами, отпусками, событиями и объявлениями.

## App shell

### Desktop

- слева фиксированный sidebar шириной около 230 px;
- сверху внутри основного окна тонкая action bar;
- основная область занимает всю высоту экрана;
- страницы не помещаются в один большой декоративный контейнер;
- таблицы, календарь и оргграф растягиваются на всю рабочую ширину.

Sidebar:

- логотип Vidnova;
- глобальный поиск с `Ctrl + K`;
- пункты: Домашняя страница, Сповіщення, Люди, Календарь, Присутності, Запити,
  Документи, База знань, Звіти;
- активный пункт подсвечивается светло-серым фоном и фиолетовой иконкой;
- настройки закреплены снизу.

Topbar:

- быстрый таймер/иконка времени;
- кнопка `Швидке додавання`;
- уведомления с badge;
- профиль пользователя справа.

### Mobile

- sidebar скрывается;
- основные разделы доступны через нижнюю навигацию;
- таблицы превращаются в карточные списки;
- тяжелые экраны, такие как календарь и оргграф, получают горизонтальный scroll,
  zoom/pan или list fallback.

## Основные экраны

### Домашняя страница

Структура как на референсе:

- верхний брендированный баннер;
- приветствие пользователя;
- быстрые действия: запрос выходного/отпуска, учет времени, объявление/опрос;
- центральная колонка: задачи и лента объявлений;
- правая колонка: остаток отпусков, недельный календарь, кто отсутствует,
  ссылки компании.

### Люди

Раздел имеет внутренние табы:

- `Люди`;
- `Команди`;
- `Організаційна структура`.

Список людей:

- верхний search bar на всю ширину;
- фильтр и переключатель вида справа;
- paginated table;
- колонки: ФИО, должность, департамент, локация, менеджер, дата начала;
- hover/detail карточка сотрудника с фото, email, департаментом, локацией и
  менеджером;
- кнопка `Новий найм` справа в title bar.

### Профиль сотрудника для HR/admin

Открывается из таблицы людей как detail screen.

Структура:

- большой брендированный баннер Vidnova сверху;
- фото сотрудника накладывается на баннер слева;
- рядом ФИО, должность и локация;
- справа кнопки предыдущий/следующий сотрудник и меню `Дії`;
- вкладки: `Особисте`, `Робота`, `Компенсація`, `Відсутності`, `Time`,
  `Документи`, `Більше`;
- основной layout: широкая левая колонка с редактируемыми секциями и правая
  summary-колонка;
- секции слева: `Особисте`, `Контакти`, `Соціальні мережі`, дальше по
  выбранной вкладке;
- каждая секция имеет кнопку `Редагувати`;
- правая колонка показывает email, телефон, дату начала, тип работы, должность,
  департамент, подразделение, локацию, стаж, менеджера;
- если данных еще нет после импорта, поля показываются как пустые/`-`, не
  заполняются демо-значениями.

### Календарь

Компания/мой календарь:

- табы `Графік` и `Календар`;
- месяц переключается стрелками;
- фильтр рядом с месяцем;
- матрица сотрудников по строкам и дней месяца по колонкам;
- отпуск рисуется горизонтальной полосой внутри строки;
- праздники/важные дни отмечаются компактными иконками;
- левая колонка сотрудников фиксированная или визуально отделена.

### Присутствие

Компания/моя посещаемость:

- заголовок `Відвідуваність`;
- переключатель `Компанія / Мої`;
- вкладки `Головна / Понаднормово`;
- справа действия `Управління проектами`, `Експорт`;
- поиск по имени на всю ширину и фильтр справа;
- toolbar месяца: `Поточний місяць`, prev/next, текущий месяц;
- кнопка `Надіслати нагадування`;
- таблица компании: ФИО, ожидаемо, отработано, понаднормово, перерва,
  оплачувана відсутність, неоплачувана відсутність, загалом відсутності,
  різниця, действия;
- экран конкретного сотрудника открывается drill-down из строки таблицы, а не
  вместо главной страницы `Присутності`.

### База знаний

Структура:

- слева дополнительный sidebar категорий;
- сверху title, поиск, кнопка управления категориями;
- центр: grid категорий/документов с счетчиками;
- категории показывают иконку, название, описание и count;
- поиск занимает отдельную строку под заголовком.

### Организационная структура

Структура:

- внутренние табы из раздела `Люди`;
- переключатель `Люди` / `Департаменти`;
- фильтр главной сущности;
- кнопки настройки и экспорта;
- большой canvas на всю рабочую область;
- карточки сотрудников в дереве: аватар, имя, должность, badge количества
  подчиненных;
- справа снизу controls: fit, fullscreen, zoom in/out, display mode.

## Визуальные правила

- фон приложения: очень светлый холодный серо-голубой;
- рабочие панели: белые, радиус до 8 px, тонкая граница;
- основной текст: темный navy;
- вторичный текст: muted blue-gray;
- primary accent: Vidnova purple;
- статусы времени: красный для переработок/ошибок, серо-голубой для недоработки,
  фиолетовый для отпусков;
- таблицы плотные, строки около 40 px;
- кнопки с иконками `lucide-react`;
- не использовать вложенные карточки для layout-секций.

## MVP UI priority

1. App shell и навигация.
2. Домашняя страница.
3. Люди: таблица + hover/detail карточка.
4. Календарь отпусков компании.
5. Присутствие сотрудника.
6. База знаний.
7. Оргструктура.
